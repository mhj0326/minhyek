Cell Tracking

Image Analysis Pipeline for Tracking Merging events of Leukemia Nuclear Bodies

This project aimed to development an automated image processing pipeline to analyze fluorescent microscopy video, focusing on detecting and quantifying fusion events between biological structures of telomeres, PML bodies, and APBs (ALT-associated promyelocytic leukemia nuclear bodies). The analysis involves fluorescently labeled puncta categorized into red, green, and yellow channels, corresponding to specific subcellular structures. The primary tool used is TrackMate 7, which provides object detection and tracking through algorithms like DoG, Hessian, and LoG. However, its default Linear Assignment Problem (LAP) tracker faced challenges in detecting complex events like merging and splitting of puncta.

To address these limitations, a custom merging detection function was developed, extracting coordinates from LAP tracker outputs, defining circular regions around detected puncta, and identifying overlaps across frames to track fusion events. Preprocessing steps, including rigid transformations, blob masking, and gamma correction, were implemented to improve detection accuracy and reduce noise. Advanced tracking methods, such as an enhanced Kalman tracker, were explored to detect merging and splitting events more effectively by incorporating penalties based on features like circularity, intensity, and track continuity.

The evaluation framework included manual annotation of merging events and quantitative comparison with the detected results to validate the pipeline's accuracy. Key challenges, such as misdetection, inconsistent circularity, and tracking failures, were mitigated by refining detection thresholds and optimizing parameters like search radius and frame gap limits. This project demonstrates the integration of advanced image analysis tools and custom algorithms to tackle the complexities of dynamic subcellular processes, enabling more accurate and robust quantification of biological phenomena.
